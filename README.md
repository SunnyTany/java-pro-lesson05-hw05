# ProductCost Project

Простое Java-приложение *для расчета стоимости товара*.

Проект демонстрирует принципы объектно-ориентированного программирования (ООП), такие как:
- **наследование**,
- **инкапсуляция**, 

также используются константы.

## 📋 Функционал

Программа выполняет два вида расчета:
- *Базовая стоимость*: Произведение количества товара на его цену.
- *Стоимость с доставкой*: Сумма базовой стоимости и фиксированной стоимости доставки.

## 🛠 Технологии

- *Язык*: Java 11+
- *IDE*: IntelliJ IDEA
- *Сборка*: Проект организован в пакет app.

## 📂 Структура проекта

    src/

        └── app/

          ├── Constants.java         # Глобальные константы (цены, валюта, данные)

          ├── Product.java           # Модель данных товара

          ├── CalcCostBase.java      # Логика базового расчета

          ├── CalcCostDelivery.java  # Логика расчета с доставкой (наследование)

          └── Main.java              # Точка входа и вывод результатов

## 🚀 Запуск

- Склонируйте репозиторий или скопируйте файлы в папку *src/app*.
- Откройте проект в *IntelliJ IDEA*.
- Запустите класс *Main.java*.

## 💻 Пример вывода

Product: abc, quota is 5 pcs., price is EUR 2.5.

Cost is EUR 12.5.

Product: abc, quota is 5 pcs., price is EUR 19.5.

Cost is EUR 19.5.

## ⚙️ Настройка

Все входные данные (название товара, количество, цена) и стоимость доставки находятся в файле Constants.java. Вы можете изменить их там для проведения новых расчетов без правки основной логики.
